# Dalal - Exploration de Design

## Réflexion Préalable

Dalal est une plateforme de bien-être mental pour la jeunesse sénégalaise et ouest-africaine. Le design doit incarner trois valeurs fondamentales :

1. **Sécurité et Bienveillance** : L'utilisateur doit sentir qu'il est en sécurité, sans jugement
2. **Authenticité Culturelle** : Respecter les réalités africaines, pas une copie occidentale
3. **Accessibilité Profonde** : Fonctionner sur connexion 2G, écrans petits, sans friction

## Approche 1 : Minimalisme Organique Africain

<response>
<probability>0.08</probability>
<text>

### Design Movement
**Afrofuturism Minimaliste** — fusion de l'esthétique africaine contemporaine avec la clarté numérique épurée

### Core Principles
- **Formes courbes et fluides** : Pas de lignes droites agressives, tout respire comme une respiration
- **Palette terracotta-sauge** : Couleurs de la terre africaine, chaudes et apaisantes
- **Typographie généreuse** : Espace blanc abondant, jamais serré
- **Micro-interactions organiques** : Animations lentes, naturelles, jamais brusques

### Color Philosophy
La palette raconte une histoire de calme progressif :
- **#F8F6F2** (Beige clair) : Fond principal — comme le sable du Sahel, accueillant
- **#4A7C6F** (Vert sauge profond) : Confiance, croissance, nature
- **#B8CBBF** (Vert sauge pâle) : Apaisement, respiration
- **#C5D5E8** (Bleu brume) : Méditation, espace mental
- **#D4B896** (Crème or) : Touche chaleureuse, humanité
- **#C4B8D4** (Lavande) : Urgence douce, jamais alarmante

### Layout Paradigm
- **Asymétrique et respirant** : Sections décalées, pas de grille rigide
- **Scroll vertical fluide** : Une seule colonne mobile-first, jamais de sidebars
- **Zones de respiration** : Padding généreux (24px mobile, 48px desktop)
- **Diagonales douces** : Utiliser clip-path pour créer des transitions organiques entre sections

### Signature Elements
1. **Ligne ondulante de respiration** : Logo animé, symbolisant le calme progressif
2. **Orbes flottantes** : Particules 3D lentes en arrière-plan, jamais distrayantes
3. **Cartes arrondies avec ombre douce** : Pas de bordures, juste de la profondeur

### Interaction Philosophy
- **Pas de clics brusques** : Chaque interaction a une transition douce (300-500ms)
- **Feedback haptique simulé** : Animations qui imitent le mouvement physique
- **Boutons qui "respirent"** : Hover state avec légère expansion et changement de couleur

### Animation
- **Respiration générale** : Toutes les animations suivent un rythme lent (4-6 secondes)
- **Entrance animations** : Éléments apparaissent progressivement, jamais instantanément
- **Scroll-triggered** : Animations qui déclenchent au scroll, créant une sensation de progression
- **Particules 3D** : Mouvement lent et organique, jamais chaotique

### Typography System
- **Titres** : Lora serif gras (h1: 32px, h2: 24px, h3: 20px) — élégant, inspire confiance
- **Corps** : Source Sans 3 régulier (16px, line-height 1.7) — ultra-lisible, professionnel
- **Citations** : Lora italic 18px — douceur, intimité
- **Jamais sous 14px** : Accessibilité sur petits écrans

## Approche 2 : Brutalism Bienveillant

<response>
<probability>0.07</probability>
<text>

### Design Movement
**Neo-Brutalism Africain** — formes épurées, typographie imposante, mais avec une chaleur humaine

### Core Principles
- **Typographie comme architecture** : Texte gros et gras crée la structure
- **Contraste maximal** : Noir profond sur blanc cassé, jamais de gris mou
- **Absence de décoration** : Seule la typographie et l'espace blanc
- **Grille visible mais flexible** : Structure claire, pas de chaos

### Color Philosophy
- **Monochrome + accent** : Noir/blanc/gris + une couleur d'accent (vert sauge)
- **#2A2A2A** (Noir doux) : Texte principal, force
- **#F8F6F2** (Blanc cassé) : Fond, chaleur
- **#4A7C6F** (Vert sauge) : Accent unique, humanité

### Layout Paradigm
- **Blocs massifs** : Sections larges, typographie énorme
- **Alignement à gauche** : Jamais centré, crée une dynamique
- **Espaces négatifs importants** : Vide intentionnel entre sections
- **Colonnes asymétriques** : Une colonne large, une colonne étroite

### Signature Elements
1. **Typographie XXL** : Titres 48px+ qui dominent l'écran
2. **Bordures épaisses** : Cadres noirs autour des sections clés
3. **Grille visible** : Lignes subtiles qui structurent l'espace

### Interaction Philosophy
- **Clics directs** : Pas de hover subtil, changement net et visible
- **Transitions brusques mais contrôlées** : 200ms, pas de courbes d'easing molles
- **Feedback textuel** : Messages clairs et directs, pas d'icônes ambiguës

### Animation
- **Minimaliste** : Seulement les transitions essentielles
- **Géométriques** : Mouvements linéaires, pas de courbes organiques
- **Rapides** : 200-300ms, jamais traînant

### Typography System
- **Titres** : IBM Plex Mono gras (h1: 48px, h2: 32px) — force, clarté
- **Corps** : IBM Plex Sans régulier (16px, line-height 1.6) — lisible, neutre
- **Accents** : Lora italic pour les citations — contraste avec le reste
- **Hiérarchie par taille** : Pas de couleur, juste du poids et de la taille

</text>
</response>

## Approche 3 : Doux Minimalisme Spirituel

<response>
<probability>0.09</probability>
<text>

### Design Movement
**Spirituel Minimaliste** — inspiré par les mandala, l'harmonie, la méditation

### Core Principles
- **Symétrie et équilibre** : Chaque élément a son équivalent
- **Cercles et courbes** : Formes organiques, jamais anguleuses
- **Palette pastel apaisante** : Couleurs douces, jamais saturées
- **Vide sacré** : L'espace blanc est aussi important que le contenu

### Color Philosophy
- **Palette pastel harmonieuse** :
  - **#F8F6F2** (Beige très clair) : Fond principal
  - **#E8E0D0** (Beige lin) : Cartes et sections
  - **#B8CBBF** (Vert sauge très pâle) : Apaisement
  - **#C5D5E8** (Bleu brume pastel) : Sérénité
  - **#D4B896** (Crème or très doux) : Chaleur subtile
  - **#C4B8D4** (Lavande très pâle) : Spiritualité

### Layout Paradigm
- **Centré et symétrique** : Chaque page a un axe central
- **Carrousels circulaires** : Navigation qui tourne, jamais linéaire
- **Mandala-inspired sections** : Éléments arrangés en cercles concentriques
- **Respiration verticale** : Sections qui s'expandent et se contractent

### Signature Elements
1. **Mandalas subtils** : Patterns géométriques en arrière-plan, très légers
2. **Cercles de couleur** : Orbes qui représentent les émotions, les états
3. **Lignes de connexion** : Filets subtils qui relient les éléments

### Interaction Philosophy
- **Doux et invitant** : Chaque interaction est une invitation, jamais une commande
- **Feedback visuel subtil** : Changements de couleur très légers
- **Transitions fluides** : Tout se transforme progressivement

### Animation
- **Respiration constante** : Éléments qui pulsent légèrement
- **Orbites** : Éléments qui tournent lentement autour d'un centre
- **Floraison** : Éléments qui s'ouvrent comme des fleurs
- **Très lent** : 5-8 secondes pour une animation complète

### Typography System
- **Titres** : Lora serif très léger (h1: 36px, h2: 28px) — élégance, légèreté
- **Corps** : Source Sans 3 très léger (16px, line-height 1.8) — aéré, lisible
- **Poésie** : Lora italic très léger — douceur maximale
- **Jamais gras** : Seulement des poids légers et réguliers

</text>
</response>

## Sélection Finale

**Approche Choisie : Minimalisme Organique Africain**

Cette approche incarne parfaitement la vision de Dalal :
- Les **formes courbes et fluides** reflètent le voyage du calme
- La **palette terracotta-sauge** est authentiquement africaine
- Les **animations organiques** créent une sensation de sécurité
- L'**asymétrie respectueuse** évite la rigidité occidentale
- Les **micro-interactions** renforcent la bienveillance

Cette approche sera appliquée rigoureusement à chaque page, chaque composant, chaque animation.
